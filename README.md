# Daniel Henrique - Visagismo

Landing page moderna e responsiva para consultoria de visagismo.

## Tecnologias
- React
- TypeScript
- Tailwind CSS
- Firebase (Firestore & Auth)
- Motion (Animações)
- Lucide React (Ícones)

## Funcionalidades
- Landing page de alta conversão
- Consultoria Online e Presencial
- Formulário de captura de leads integrado ao Firebase
- Design responsivo e animações fluidas
